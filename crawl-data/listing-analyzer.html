Overview

Stop guessing why a listing isn't selling.

What it does

Listing Analyzer pulls every listing element — title, tags, description, images, videos, materials — and grades each one against Etsy's best practices. You get a letter grade plus actionable feedback on what to fix first.

Why it matters

Most listings look fine on the surface but quietly miss search visibility from a few fixable details. Audit any listing — yours or a competitor's — and you'll know exactly which levers to pull.

Getting started

How to use it.

  1. 01

    Enter a listing URL or ID

    Paste the full Etsy listing URL or just the listing ID number. Works for any public Etsy listing.

  2. 02

    Wait for the analysis to load

    We fetch fresh data from Etsy and grade every element of the listing — title, tags, description, images, videos, materials. Takes a few seconds.

  3. 03

    Review the optimization grade

    Start with the overall letter grade, then drill into per-section scores to see which elements are pulling the grade down.

  4. 04

    Apply the feedback

    Expand each section to see specific suggestions. Tackle the lowest-scoring sections first — small fixes across multiple areas usually give the biggest grade lift.
